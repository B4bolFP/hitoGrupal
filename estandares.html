Estándares


En un principio el desarrollo de las páginas web era caótico por la razón que había que crear una versión de cada sitio web para cada navegador. Para solucionar esto en 1997 se lanzó una iniciativa para lograr la accesibilidad web y que se siguieran una serie de estándares.
A raíz de entonces se fueron desarrollando diferentes estándares logrando que cada página web se vea correctamente independientemente del navegador o dispositivo. Los estándares web más conocidos son:
  • HTML (HyperText Markup Language), para definir la estructura de los documentos.
  • XML (eXtensible Markup Language) sirve de base para un gran número de tecnologías.
  • CSS (Cascading Style Sheets) permite asignar estilos para la representación de los documentos.
  • Javascript permite otorgar dinamismo y funcionalidad.
