CSS y SCSS


  • CSS es un lenguaje de script lanzado el 17 de diciembre de 1996 usado para diseñar el estilo de las páginas web.

  • SCSS es una versión más avanzada de CSS lanzada el 28 de noviembre de 2006 y fué creada por Hampton Catlin y desarrollado por Chris Eppstein y Natalie Weizenbaum. Incluye cosas como la posibilidad de incluir variables para acortar líneas de código, importar otros archivos SCSS, personalizar Bootstrap y por último permite usar sintaxis anidada lo que quiere decir que puedes editar el estilo de cosas mucho más específicas.
